# ppt-design-skill (ppt-디자인구축)

3교시 PPT 방법① **게이트형 스킬**. getdesign.md에서 디자인 시스템을 골라 우리 회사 톤으로 바꾸고,
엑셀 샘플로 PPT를 만든 뒤 — 그 흐름을 **재사용 스킬**로 굳힌다.

## 흐름 (5단계)

1. **디자인 고르기** — getdesign.md 직접 방문 → 선택 → Copy → 붙여넣기
2. **우리 것으로** — 색·폰트·톤을 우리 회사로 (+ 컴플라이언스 내장)
3. **한 장 테스트** — 피드백 루프로 한 장씩 다듬기
4. **본격 사용** — 엑셀 → 일관 톤 PPT 자동 생성
5. **스킬로 등록** — "이 과정을 스킬로 만들까요?" → 재사용 자산화

> 1~4단계는 처음 한 번, 5단계로 스킬화하면 다음부턴 한 줄.

## 구조

```
ppt-디자인구축/
├── SKILL.md          # 스킬 정의 (게이트형, 자동 실행 금지)
└── sample-data/      # 2교시와 동일한 엑셀 6종 (엑셀 → PPT 실습용)
```

> 같은 엑셀 → 2교시는 HWP, 3교시는 PPT로. 데이터는 그대로, 포맷만 바뀐다.

## 사용

채팅창에 트리거 입력:
- `PPT 디자인 구축하자` ⭐
- `디자인 시스템 만들기` / `getdesign으로 PPT 디자인`

## 방법① vs 방법②

- **방법①(본 스킬)**: 디자인 시스템을 **스킬로 등록** — 빠르고 비개발자 친화.
- **방법②**: 디자인을 직접 만들어 **코드(토큰·템플릿)로 고정** — 일관성·대량 생산.

## 의존성

- 글로벌 스킬 `pptx` (.pptx 생성)
- getdesign.md (디자인 시스템 소스)
