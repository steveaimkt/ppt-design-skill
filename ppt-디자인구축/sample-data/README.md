## 실습 매핑
- **HWP(2교시)**: 물류·품질·생산 — 위험/미조치/저조가 명확해 "현황 → 시사점 → 조치요청" 공문 흐름이 잘 나온다.
- **PPT(3·4교시)**: 영업·MD·마케팅 — 주인공 스토리(급성장·신상·효자 캠페인)가 있어 "결론 먼저" 제안서가 잘 나온다.

> 엑셀 미지참 수강생은 본 6종 중 본인 부서에 가까운 1개를 골라 실습.
